News Parser Bot

News Parser Bot — це Telegram-бот для автоматичного збору та відправки актуальних новин з обраних джерел. Бот підтримує категоризацію новин, інтеграцію з AI для генерації контенту та зручний інтерфейс взаємодії. News Parser Bot моніторить новинні ресурси та надсилає користувачам найважливіші події. Можливості включають автоматичний парсинг, фільтрацію новин за категоріями, генерацію коротких описів через AI та підтримку безперервної роботи бота через keep-alive механізм.

Можливості
Функція	Опис
Автоматичний парсинг	Парсинг новин з RSS-джерел та веб-сайтів
AI-генерація контенту	Створення коротких описів новин та узагальнень
Telegram інтерфейс	Підтримка команд для взаємодії з користувачем
Категорії новин	IT, спорт, політика, економіка
Keep-alive	Механізм для безперервної роботи бота
Валідація URL та обробка зображень	Перевірка посилань і підготовка зображень для повідомлень
Технології
Компонент	Використання
Python 3.8+	Основна мова програмування
python-telegram-bot	Робота з Telegram Bot API
requests	HTTP-запити для отримання новин
BeautifulSoup4	Парсинг HTML-контенту
Flask	Веб-сервер для keep-alive механізму
Структура проєкту
tg_bot/
├── run.py              # Головний файл запуску бота
├── handlers.py         # Обробники команд та повідомлень користувачів
├── ai_generate.py      # Модуль для генерації контенту за допомогою AI
├── news_scraper.py     # Парсер новин з RSS фідів та веб-джерел
├── config.py           # Конфігураційні налаштування та змінні
├── keep_alive.py       # Flask сервер для підтримки активності бота
├── Procfile            # Конфігурація для запуску процесів
└── pon.txt             # Додаткові дані

Опис модулів
Модуль	Опис
run.py	Ініціалізація та запуск бота, підключення всіх модулів
handlers.py	Обробники команд користувача (/start, /help, /news)
ai_generate.py	Робота з AI API для генерації контенту та описів новин
news_scraper.py	Збір та парсинг новин з різних джерел, валідація URL, обробка зображень
config.py	Зберігання токенів, API ключів, джерел новин та налаштувань
keep_alive.py	Flask сервер для підтримки активності бота

Встановлення
Крок 1: Клонування репозиторію
git clone https://github.com/daniyil64345/tg_bot.git
cd tg_bot

Крок 2: Встановлення залежностей
pip install -r requirements.txt

Крок 3: Налаштування

Створіть файл .env у кореневій директорії:

TELEGRAM_TOKEN=your_telegram_bot_token
API_KEY=your_ai_api_key


Як отримати токен Telegram: знайдіть @BotFather у Telegram, надішліть команду /newbot та дотримуйтесь інструкцій, потім скопіюйте токен у .env.

Крок 4: Запуск
python run.py
